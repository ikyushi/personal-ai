# simplesent(ai)

A personal AI assistant to help out with your day to day tasks.

## Features

- Date and time functions
- Weather
- News
- And much more!

## Installation

- Clone the repository
- Install dependencies
- Run `node .`
- Follow setup instructions (in-app)
- Done!
